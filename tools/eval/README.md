# Evaluation Tools

Run these from the project root unless noted otherwise.

- `python tools/eval/run_v6_hybrid_benchmarks.py`
- `python tools/eval/check_gpu_torch.py`
- `python tools/eval/evaluate_v6_adapter.py`
- `python tools/eval/evaluate_v6_adapter_transformers.py`
- `python tools/eval/diagnose_v6_adapter.py`
- `python tools/eval/test_finetuned_math_assistant.py`

The V6 hybrid benchmark is local and tool-assisted. It is not an official
GSM8K/MATH/AIME leaderboard score.
